## 컨셉 (1줄)

> **세로 모바일 타워 디펜스. 웨이브 사이에 카드를 픽해 덱을 빌드하는 로그라이트.**

## 타겟

- **플랫폼**: iOS / Android (Capacitor 래핑, 웹은 dev only)
- **기기 자세**: 세로 (한 손 플레이)
- **한 판 길이**: 3~5분
- **타겟 유저**: 출퇴근/짜투리 시간 모바일 게임 유저
- **참고 장르**: Tower Defense + Roguelite Card Pick

---

## 코어 게임 루프

```
매치 시작 → 웨이브 진행 → 카드 픽 → 다음 웨이브 → ... → 보스 → 결과
```

- 매치당 5 + 1(보스) = **6 웨이브**
- 웨이브 클리어 시 카드 3장 중 1장 선택
- 베이스 HP 0 = 패배 / 보스 처치 = 승리

---

## 플레이어 행동

| 행동 | 입력 |
|------|------|
| 카드 픽 | 카드 탭 |
| 타워 배치 | 카드를 슬롯에 드래그 |
| 타워 업그레이드 | 타워 탭 후 업그레이드 버튼 |
| 일시정지 / 재개 | 우상단 버튼 |

→ **이게 전부.** 입력 어휘를 늘리지 않는다.

---

## 타워 (3종)

| 타워 | 사거리 | DPS | 특징 | 시작 비용 |
|------|--------|-----|------|----------|
| **Sniper** | 길다 | 중 | 단일 타겟, 느린 발사 | 100 |
| **Cannon** | 중 | 중 | AOE 반경 60px, 느린 발사 | 150 |
| **Frost** | 짧다 | 약 | 적 이속 -40% (2초) | 80 |

→ 정확한 수치는 5주차 밸런싱에서 확정.

## 적 (3종)

| 적 | HP | 속도 | 특징 |
|----|----|------|------|
| **Grunt** | 10 | 1.0x | 표준 |
| **Runner** | 5 | 1.8x | 빠르지만 약함 |
| **Tank** | 30 | 0.6x | 느리지만 단단함 |

## 카드 픽 시스템

웨이브 클리어 시 다음 3개 카테고리에서 무작위로 3장 추출, 1장 선택:

| 카테고리 | 효과 예시 |
|---------|----------|
| **New Tower** | "Sniper 카드 1장 (배치 가능)" |
| **Upgrade** | "랜덤 타워 1개 +1렙" |
| **Passive** | "모든 타워 데미지 +10%" |

→ 등급(Common/Rare/Epic) 도입은 MVP 이후 검토.

## 맵

- 720 x 1280 세로 화면
- 직선 또는 단순 곡선 경로 (적이 따라감)
- 경로 양옆에 타워 슬롯 6~8개 고정 배치

```
┌─────────────────┐
│ HUD (HP, Wave)  │
├─────────────────┤
│      [start]    │
│        │        │
│   □    │    □   │  ← 타워 슬롯
│        │        │
│   □    │    □   │
│        │        │
│   □    │    □   │
│        │        │
│      [base]     │
├─────────────────┤
│  Pause | Speed  │
└─────────────────┘
```

## 매치 흐름

```
[Title]
   ↓ "Play" 탭
[Match Setup]  (시작 시 보유 카드 3장 자동 지급)
   ↓ "Start" 탭
[Match]
   ├─ Wave 1 → Card Pick (3장 중 1장)
   ├─ Wave 2 → Card Pick
   ├─ Wave 3 → Card Pick
   ├─ Wave 4 → Card Pick
   ├─ Wave 5 → Card Pick
   └─ Boss Wave → 결과
   ↓
[Result] (Win / Lose) → "Retry" or "Title"
```

## 기술 스택

| 레이어 | 도구 |
|--------|------|
| 게임 엔진 | Phaser 4 (TypeScript) |
| 빌드 | Vite |
| 모바일 패키징 | Capacitor (iOS / Android) — 6주차 |
| 저장소 | localStorage (MVP) → Firebase (향후) |
| 그래픽 | placeholder 도형 → 본인 제작 / CC0 (출시) |
| 사운드 | (없음) → freesound.org / CC0 (출시) |

---

## 명시적 비목표 (MVP 단계에서 안 만든다)

- 메타 진행 (카드 영구 수집, 영구 업그레이드)
- 다중 맵 / 챕터
- 계정 / 백엔드 / 클라우드 저장
- 광고 / IAP
- 리더보드 / 멀티플레이
- 정식 아트 / BGM / 효과음
- 다국어 (한국어 하드코딩)

---

## 위험과 가정

| 위험 | 완화 |
|------|------|
| 카드 픽이 의미 있는 의사결정이 안 됨 | 5주차 자체 검증 + 친구 플레이테스트 |
| 모바일 성능 문제 | placeholder 단계는 부담 적음, 6주차 실기기 확인 |
| 본인 IP 미검증 상태에서 시간 투입 | MVP는 검증 도구 — 재미없으면 피벗 OK |
| Phaser 4 신규 버전 이슈 | API는 Phaser 3과 호환, 빌드는 통과 확인됨 |
