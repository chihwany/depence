# 01. 장르 시스템 분석

> Raid Rush(Panteon, 2022)가 속한 **타워 디펜스 + 카드 픽 로그라이트 + 메타 진행** 장르의 일반적인 시스템 구조 정리.
>
> 이 문서는 Raid Rush를 복제하기 위한 것이 아니라, **같은 장르의 게임이 어떤 시스템 레이어로 구성되는지** 학습하기 위한 것이다. 본인 게임은 이 구조를 참고하되 고유한 컨텐츠로 채운다.

---

## 시스템 3층 구조

장르의 모든 게임은 다음 3개 레이어로 분해할 수 있다:

```
┌──────────────────────────────────┐
│  C. 인게이지먼트 / 수익화 레이어    │  ← 장기 리텐션 (이벤트, 배틀패스, 광고)
├──────────────────────────────────┤
│  B. 메타 진행 레이어               │  ← 매치 사이의 성장 (덱, 화폐, 카드 수집)
├──────────────────────────────────┤
│  A. 코어 게임 루프 (인매치)        │  ← 한 판 (3~5분), 재미의 본질
└──────────────────────────────────┘
```

**MVP에서는 A만 만든다.** B와 C는 "코어가 재밌다"가 검증된 후에 붙인다.

---

## A. 코어 게임 루프 (한 판, 3~5분)

이 장르의 코어 루프는 보통 다음 구조다:

```
[매치 시작]
   ↓
[웨이브 1] ─→ [적 스폰] ─→ [타워 자동 사격]
   ↓                            ↓
   └─ 적이 베이스 도달 시 HP 감소 ┘
   ↓
[웨이브 클리어]
   ↓
[카드 3장 중 1장 선택]
   (새 타워 배치권 / 기존 타워 강화 / 패시브 효과)
   ↓
[웨이브 2] ... 반복
   ↓
[보스 웨이브] → 처치 시 [매치 승리]
   ↓
[베이스 HP 0] → [매치 패배]
```

### 핵심 의사결정 포인트

플레이어가 "고민"하는 순간이 매치당 몇 번 있는가가 재미를 결정한다:

1. **카드 픽** — 어떤 빌드를 갈지 (덱빌딩의 묘미, 가장 중요)
2. **타워 배치** — 어디에 둘지 (공간 퍼즐)
3. **타이밍** — 업그레이드할지 새 타워 둘지 (자원 분배)

### 플레이어 행동(Verbs)

플레이어가 손가락으로 **할 수 있는 일의 총 목록**:

- 카드 탭 (선택)
- 카드 드래그 (배치)
- 타워 탭 (업그레이드 메뉴)
- 일시정지/재개

→ 이게 전부다. 모바일 게임은 입력 어휘가 좁을수록 좋다.

### 인매치 카드 시스템의 두 가지 방식

이 장르에서 흔한 패턴 두 가지:

| 방식 | 설명 | 예시 |
|------|------|------|
| **사전 덱 + 인매치 추첨** | 매치 전 8장 덱 구성 → 매치 중 무작위 등장 | Clash Royale 스타일 |
| **인매치 카드 픽** | 매치 중 웨이브 클리어 후 3장 중 1장 선택 | Vampire Survivors / Slay the Spire 스타일 |

Raid Rush는 두 가지를 **혼합** 사용. MVP에서는 단순화를 위해 **인매치 카드 픽만** 구현 권장.

---

## B. 메타 진행 레이어 (매치 사이)

| 시스템 | 역할 | MVP 포함? |
|--------|------|-----------|
| 화폐 (소프트) | 인매치 보상, 메뉴에서 사용 | △ 단순화 |
| 화폐 (하드) | 스테이지 클리어 보상 | × |
| 화폐 (프리미엄) | IAP / 광고 보상 | × |
| 카드 수집 | 새 타워 영구 해금 | × |
| 카드 업그레이드 | 중복 카드로 강화 (가챠 패턴) | × |
| 배틀 덱 구성 | N장 카드 선택 후 매치 입장 | × |
| 플레이어 레벨 | 컨텐츠 게이팅 | × |
| 챕터/스테이지 진행 | 새 맵, 새 적 등장 | × |

→ MVP는 메타 시스템 거의 다 빼고 **"매치를 다시 플레이할 동기"만 남긴다** (다음 시도, 다른 카드 조합 — 그것만으로).

---

## C. 인게이지먼트 / 수익화 레이어

장르 표준 패턴 (MVP 대상 아님, 출시 단계에서 도입):

| 시스템 | 패턴 |
|--------|------|
| 데일리 보상 | 7일 연속 출석 점증 |
| 데일리/위클리 미션 | "오늘 3매치 승리" 류 |
| 배틀패스 | 시즌(4~8주) 트랙 진행 |
| 시즌 이벤트 | 한정 카드/스킨 |
| 리더보드 / 아레나 | 트로피 기반 매칭 |
| 광고 (Rewarded) | 무료 박스, 더블 리워드, 부활 |
| IAP | 잼 패키지, 스타터 팩, 배틀패스, 한정 번들 |
| 카드 박스 (가챠) | 등급별 확률 (Common/Rare/Epic/Legendary) |

**수익화 모델 = 광고 + IAP 하이브리드** (장르 표준)

---

## 학습 포인트

1. **A 레이어가 재미없으면 B, C 다 무의미** — MVP는 A에만 집중
2. **B, C는 리텐션 도구** — 재미를 *늘리는* 게 아니라 *유지하는* 도구
3. **카드 픽은 로그라이트 요소** — 매번 다른 빌드가 가능해서 반복 플레이 동기 형성
4. **타워 디펜스 + 카드 픽** 조합은 비교적 새로운 하위 장르 — 잘 만들면 차별화 가능
