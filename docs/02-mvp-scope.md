## "재미 있다"의 판단 기준

5주차 자체 평가 시 다음 3개 질문에 모두 **YES**면 통과:

1. **한 판이 끝났을 때 "한 판 더" 누르고 싶은가?**
2. **카드 픽에서 고민이 되는가?** (모든 카드가 명백히 좋거나 나쁘면 X — 의사결정이 없는 것)
3. **다른 빌드(다른 카드 조합)로 시도해보고 싶은가?**

→ 3개 다 YES면 다음 단계(메타 시스템) 진행.
→ 하나라도 NO면 코어 튜닝 더 (밸런싱, 카드 효과 재설계).

## 밸런싱 수식 — 현재 적용된 값

> 원본 템플릿(작성 시점) → **실제 게임에 적용된 값**

### 타워 DPS 곡선

```
업그레이드 데미지 = base_damage × 1.5^(level - 1)
사거리/발사주기는 안 변함
```

예: Sniper L1 = 8 dmg, L2 = 12, L3 = 18, L4 = 27, L5 = 40

### 적 곡선 (수식 → 명시 테이블로 변경)

```
원래 계획: HP = base × 1.25^(N-1) — 자동 스케일
실제 적용: WAVES 배열에 명시적 정의 (4→6→10→15→20→보스)
```

이유: 명시적 정의가 디자인 의도(러너/탱크 등장 시점)를 통제하기 쉬움. 원본 수식은 미래 챕터 확장 시 참고용.

### 데미지 부스트 카드

```
Power Up 카드: 전체 타워 데미지 × 1.25 (중첩 가능)
3장 중첩 시: 1.25^3 ≈ 1.95 (거의 2배)
```

### 베이스 수치

```
maxHp = 15
auto-defense damage = 5
auto-defense range = 140 (= 2 × cellSize 70)
auto-defense fireRate = 700ms
```
